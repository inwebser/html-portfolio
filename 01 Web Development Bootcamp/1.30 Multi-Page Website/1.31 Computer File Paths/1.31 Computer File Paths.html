Direct the Computer to a Particular File or a Folder.

Two types of File Paths, viz, Absolute & Relative.

  1. Absolute File Paths:

    2. Path that is relative to the root of a computer, say, C drive in case of Windows. C here is the root directory.
    E.g, C/Users/Junaid/Downloads/image.jpg
    C is the root and image.jpg is the location.

  2. Relative File Paths:

    3. Path that is relative to the starting point, say, index.html
