Doctype HTML.

(HTML Doctype tells us the version the html file was written in.)
The structure is like this:

  1. HTML tag where the code is written.
  2. Inside of the HTML tags are
    • Head element, where you include important information that is not visible to a user.
      For example, meta tag which takes care of character set like UTF-8 and website title.
    • Body element, inside which all of the website content goes, like text, images, titles, links etc.
