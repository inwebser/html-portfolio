Unordered List using (ul).

For example:

Foods that I like:

  • Lean meat.
  • Chicken breasts.
  • Haakh saag.
  • Palak.
  • Beef/Mutton Barbeques.

Same list can be ordered using (ol):

  1. Lean meat.
  2. Chicken breasts.
  3. Haakh saag.
  4. Palak.
  5. Beef/Mutton Barbeques.

Same list can started from a particular point,
for example the list starts from 5, we will add 5 to the (ol):

  1. Lean meat.
  2. Chicken breasts.
  3. Haakh saag.
  4. Palak.
  5. Beef/Mutton Barbeques.
